# Savings Manager

A web application designed to help savings groups manage members, track contributions, and organize payouts with clarity and efficiency. Built with a modular structure for easy growth and hosted via GitHub Pages.

## 🚀 Live Demo
[View the app here](https://csgapp.github.io/savingsmanager/)

## 📂 Features
- **Member Portal**: Secure login, history tracking, and personalized dashboard.
- **Admin Tools**: Add and manage members, oversee group activity, and configure payouts.
- **Share‑Out Tracking**: Clean tables and forms to calculate and distribute savings.
- **Reports & Charts**: Visual insights into group performance.
- **Responsive Design**: Works across desktop and mobile devices.
- **Password Management**: Setup and reset flows for user accounts.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Hosting**: GitHub Pages
- **Version Control**: Git + GitHub
